◆ node_findneighbor()

void node_findneighbor ( int  i,
int  j 
)
  • 对第(i,j)个节点寻找相邻的网格， 从(i,j)个网格逆时针数第一个存在的网格开始记录，逆时针顺序。
  • 对第(i,j)个节点寻找相邻的节点， 从(i-1,j)开始逆时针算起第一个存在的节点开始记录，逆时针顺序。
  • 计算各边的（部分）边长和外法向量
参数
i节点的行编号
j节点的列编号
